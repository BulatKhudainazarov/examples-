		Блоч. и стр. элементы.
Блочные: p, ul, ol ,h1-h6, article, section, blockquote
Строчные: a, em, strong, q, abbr

Блочные эл. нужны для структурирования часей стр. разделяя содер.
на маленькие блоки

Строчные нужны для выделения текста из общей картины (смена цвета,
гиперссылки...)

все самозакрыв. эл. СТРОЧНЫЕ

		Иерархия в HTML

<p>Сэр <strong>Алекс Фергюсон</strong> однажды сказал о Филиппо Индзаги: 
<q>Этот парень должен был родиться в положении вне игры</q>.</p>

элемент <strong> задаёт слова «Алекс Фергюсон» более важными;
<q> отмечает его цитату об Индзаги.

Нужно выбирать HTML чтобы они соответ. значению HTML, если текст важный 
нужно добав. важность этой строке и слову. 
Не нужно ориент. на внеш. вид стр.

<p> — родительский элемент для <strong> и <q>;
<strong> и <q> — дочерние элементы для <р>;
<strong> и <q> — братские элементы.
		
		Семантика

Типичная веб стр. выглядит так: 

Это все структурные блоки

<header> первый эл.: логотип и слоган
<nav> второй навигация, ссылки по сайту
<h1> третий эл. заголовок стр
<article> основное содер. но больше для статей
<main> наст. основное сод.
<footer> после. эл. ссылки на контакты и так далее
<div> ЕСЛИ не один из этих блоч не подходит ИСПОЛЬЗУЕМ

<p> для абзацев;
<ul> для (неупорядоченных) списков;
<ol> для (упорядоченных) списков;
<li> для отдельных пунктов СПИСКА!!;
<blockquote> для цитат.
<div> ЕСЛИ не один из этих блоч не подходит
